# lassaVacc_HCWs
Model evaluating health impacts of vaccinating Nigerian healthcare workers against Lassa fever. Composed of 6 scripts and a directory of data input files. 

This code was developed, tested and run in R v.4.5.1. All packages required to run the code are listed in housekeeping.R.

Running this code reproduces all quantitative results and figures in the manuscript. Please note that, due to data limitations, one previously generated input dataset representing age- and sex-stratified LASV zoonosis incidence at the Admin1 level (data/df_incidence_2019_allAges.Rdata) must be generated following https://github.com/drmsmith/lassaRiskVac (infections_foi.R), and one data output from the present repository (outputs/df_vaccOutcomes_annual.Rdata) is not pre-loaded and must be generated by running hospital_model.R.

### (1) housekeeping.R
Base functions and labels

### (2) nigeria_data.R
NCDC case data time series and summary statistics

### (3) hospital_model_general_inputs.R
General model inputs; this loads from the /data directory, including data from a literature review of Lassa fever CFR in HCWs (review_data_extraction.xlsx), state-level population estimates (df_population.Rdata), life expectancy data (df_un_le_nigeria_2025.csv) and decision tree parameters (params_montecarlo.Rdata)

### (4) hospital_model.R
The model, simulates outcomes and vaccine impact

### (5) hospital_model_summarize.R
Summarizes simulation results

### (6) hospital_model_tables_plots.R
Data tabulation and visualisation
